# lab-cn-portal

## 项目简介

`lab-cn-portal` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何单个域名或具体网站，旨在提供一个结构化的存放空间，用于整理和分享各类静态页面资源。

## 目录说明

```
lab-cn-portal/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库入口页面
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面，页面之间相互独立，无强依赖关系。
- `assets/`：存放页面通用的样式、脚本和媒体文件，便于复用和维护。
- `index.html`：仓库的默认入口页，可提供页面链接的导航或列表。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 每个页面通常包含完整的 HTML 结构，可直接在浏览器中打开查看。
- 页面内容可能涉及不同主题或用途，无统一主题约束。
- 归档的页面可能不定期更新、替换或移除，以保持仓库内容的整洁。
